# PhotoMint 图片压缩桌面软件

https://photomint.online

### 特色
1. 简单，高效
2. 拖入批量处理
3. 自动保存

基于 [tauri 2.0](https://tauri.app/) 和 [squoosh](https://github.com/jamsinclair/jSquash)
